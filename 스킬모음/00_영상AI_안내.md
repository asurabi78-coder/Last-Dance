## fal-ai-community (제품·광고 영상의 핵심) — `영상_AI/fal-ai-community/`
fal.ai의 genmedia 엔진 기반. 스마트스토어 상세페이지 영상에 가장 적합.

| 스킬 | 용도 |
|---|---|
| **ugc** ★ | 크리에이터형 제품 데모·언박싱·비포애프터·후기·세로형 숏폼 |
| **commercial** ★ | 제품 리빌·이커머스 배치·라이프스타일 광고·배경 교체 |
| **marketing** | 캠페인 단위(런칭 키트, 채널별 변형, 광고 패키지) |
| **genmedia / genmedia-workflow** | 핵심 CLI 엔진 · 생성→편집→자막→납품 파이프라인 |
| **cinematography** | 카메라 무빙·조명·렌즈·색보정 등 시네마틱 연출 |
| **fal-prompting / fal-models-catalog / fal-recipes** | 프롬프트 작성 · 모델(Kling 등) 카탈로그 · 레시피 |
| **storytelling / character-design / fan-cam / model-routing** | 스토리·캐릭터·팬캠·모델 라우팅 |

## 추천 제작 흐름 (스마트스토어 제품 영상)

1. **컷 생성** → `fal/commercial` 또는 `fal/ugc` 로 제품 사진을 사용 장면 영상으로 (Kling 모델 추천: `fal-models-catalog`)
2. **연출 보강** → `fal/cinematography` 로 카메라·조명 디렉션
3. **음악·내레이션** → `MiniMax/minimax-music-gen`, `minimax-multimodal-toolkit`(음성)
4. **자막·텍스트·로고 합성** → `remotion` 으로 상세페이지용 완성본 편집
5. **파이프라인 자동화** → `fal/genmedia-workflow`

---

## 꼭 알아둘 점

- 이 스킬들은 **API 키가 필요하고 대부분 유료**입니다: fal.ai(genmedia), MiniMax. remotion은 무료(렌더는 로컬/클라우드).
- 실제 "제품을 손으로 사용하는" 자연스러운 동작은 **Kling 계열 모델**이 가장 안정적입니다(`fal-models-catalog`에서 선택).
- OpenAI Sora는 별도 접근(Sora API)이 필요하며 이 저장소엔 포함돼 있지 않습니다.
- fal 스킬들은 `genmedia` CLI에 의존합니다 → `genmedia init` 1회 설치 후 사용.
