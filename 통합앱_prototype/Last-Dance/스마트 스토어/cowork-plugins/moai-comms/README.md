# moai-comms

직장 커뮤니케이션·소프트스킬 플러그인 — 보고·설명, 회의 진행, 피드백 주고받기, 갈등·소통빌런 대응, 1:1 면담·설득·협상.

직장에서 매일 부딪히는 **사람 사이의 소통**을 돕습니다. 상사에게 결론부터 꽂히게 보고하는 법, 산으로 가는 회의를 결론까지 끌고 가는 진행, 부정적 피드백을 소화하고 건설적으로 전달하는 법, 까다로운 동료와 거리를 지키며 일하는 법, 면담·요청·정중한 거절·협상까지 2026년 한국 직장 기준으로 안내합니다. 기술이 아니라 **태도와 구조**로 풀어가는 소프트스킬 모음입니다.

## 스킬

| 스킬 | 설명 | 상태 |
|------|------|:----:|
| [report-speak](./skills/report-speak/) | 결론 먼저·두괄식 보고, 상사 유형별 보고 방식, 상대 뇌에 꽂히는 설명, 핵심 요약 | ✅ |
| [meeting-facilitator](./skills/meeting-facilitator/) | 아젠다 설계, 산으로 가는 회의 방지, 의사결정 끌어내기, 회의록·후속 액션 | ✅ |
| [feedback-loop](./skills/feedback-loop/) | 부정적 피드백 소화, 피드백 회피 극복, 건설적 피드백 전달(SBI), 중간 피드백 요청 | ✅ |
| [conflict-handler](./skills/conflict-handler/) | 소통빌런 유형별 대응, 정중한 거절, 감정 분리, 직장 관계 거리 두기 | ✅ |
| [negotiation-1on1](./skills/negotiation-1on1/) | 1:1 면담 준비, 설득 구조 설계, 요청·부탁·정중한 거절, 협상 태도 | ✅ |

## 사용 예시

```
내일 팀장님한테 프로젝트 지연 보고해야 하는데 어떻게 말해야 깔끔할까?
```

```
이번 회의가 자꾸 산으로 가는데 아젠다랑 진행을 어떻게 잡지?
```

```
자꾸 떠넘기는 동료한테 기분 안 상하게 거절하는 법 알려줘
```

## 주요 워크플로우 체인

```
중요한 보고 준비
  report-speak(결론·구조 설계) → conflict-handler(까다로운 상사 대비) → meeting-facilitator(보고 회의 진행)

피드백 면담 준비
  feedback-loop(피드백 정리) → negotiation-1on1(면담 대화 설계)

까다로운 요청 풀기
  conflict-handler(상황·유형 진단) → negotiation-1on1(요청·거절 구조)
```

## 다른 플러그인과의 경계

| 비슷해 보이지만 다른 영역 | 사용해야 할 스킬 |
|---|---|
| 공식 인사 면담·성과평가·평가 피드백 | `moai-hr/performance-review` |
| 보고서·기획서·이메일 글쓰기·문서 작성 | `moai-office` / `moai-content` |
| 마케팅 카피·콘텐츠 글쓰기 | `moai-content` 계열 |
| 개인 목표·습관·회고 등 자기관리 | `moai-productivity` |
| 영업 제안·계약 협상 문서 | `moai-business` |

## 한국 직장 커뮤니케이션 특화

- **2026년 한국 직장** 맥락(수평·하이브리드 근무, 메신저 보고)을 반영한 화법
- **상사 유형별** 보고·대응 프레임 (디테일형·결론형·관계형 등)
- **구조로 푸는 소프트스킬** — 의지가 아니라 말의 순서·요청 구조를 바꿔 해결
- **감정과 사안 분리** — 갈등을 인격이 아니라 행동·사안 단위로 다루기

## 설치

Settings > Plugins > cowork-plugins에서 `moai-comms` 선택
